Big Jon's VIM setup

Note: This is a personal repo designed to what "I" prefer. Feel free to clone/fork to your needs.

Install:

cd ~ && mkdir .vim && cd ~/.vim

git clone git@github.com:bigjonwilliams/bigjon-vim.git .

git submodule init && git submodule update
